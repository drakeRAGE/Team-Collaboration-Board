### Description
This is a full stack application for Team Collaborative Board, and this is made with MERN Stack as a clone of Asana for productive team tasks management and better task scheduling.

## Installation

Install project with npm

```bash
  cd client
  npm install 
  npm run dev
```
    
## Run application

To run application , write command : 

```bash
  cd ../server
  npm install
  npm start
```

Access the website:
Open your browser and navigate to https://team-collaboration-board-amber.vercel.app

## Environment Variables

To run this project, you will need to add some environment variables to your .env files. Check the env examples (.env.example) files in client and server dir and create same .env files with your details.
